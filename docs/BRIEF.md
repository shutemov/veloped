# Бриф: личный трекер велопоездок

## Суть продукта

**Личный трекер велопоездок** — мобильное приложение для записи маршрутов, дистанции и времени в пути без облака и соцсетей. Данные хранятся только на устройстве. При желании можно использовать и для пеших прогулок или пробежек.

В двух словах: *аналог Strava для одного пользователя, офлайн, с открытым стеком на Android.*

---

## Для кого

Для себя: один пользователь, одно устройство. Никаких аккаунтов и регистраций — данные только на этом устройстве.

---

## Зачем

- Видеть маршруты на карте и историю поездок.
- Знать дистанцию и время в пути по каждой поездке.
- Не зависеть от подписок и не отдавать данные сторонним сервисам вроде Strava.
- Собирать статистику локально (неделя, месяц) при желании.

---

## Что не входит (границы скоупа)

- Нет аккаунтов, синхронизации между устройствами и облачного бэкенда.
- Нет соцфункций: друзья, лайки, комментарии, лидерборды.
- Нет сегментов в стиле Strava и интеграций со Strava / Garmin в MVP.
- Нет платных картографических API — только открытые (например, OpenStreetMap).

---

## Платформа

- **Android** — приоритетная платформа, все фичи реализуются в первую очередь под неё.
- **iOS** — возможное развитие позже при сохранении того же стека (Expo/React Native).

---

## Итог

Приложение для личного учёта велопоездок (и при желании других активностей): запись трека в реальном времени, карта с маршрутом, дистанция, время, история — всё локально и средствами открытого ПО, без облака и соцсетей.
